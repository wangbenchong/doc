# SpaceSniffer软件

可以用它来搜索各个磁盘的文件占用情况，很直观

记得它的快捷方式设置成有管理员权限（右键属性-兼容性里），这样扫描的时候不会出现无法访问的报错。

# Windows自带磁盘清理

搜“磁盘清理”即可

# pagefile.sys瘦身

如果放任不管，这个文件可能会变得很大。

- 右键”此电脑”->属性->高级系统设置->性能->设置(S)...->高级->虚拟内存->更改(C)...
- 勾选“自定义大小”，初始大小设为：2048，最大值设为：4096，然后记得点 设置(S) 按钮，否则不会保存
- 一路确定，会提示重启电脑，重启后文件会瘦身。

# Edge浏览器缓存清理
